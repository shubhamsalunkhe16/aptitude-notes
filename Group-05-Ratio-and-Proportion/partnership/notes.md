# Partnership - Aptitude Tricks Cheatsheet

---

## ALL FORMULAS AT A GLANCE

### Simple Partnership

```
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│  All partners invest for the SAME duration                          │
│  Profit ratio = Capital ratio                                       │
│                                                                     │
│  A : B = Cₐ : Cᵦ                                                    │
│  A's share = Total Profit × Cₐ / (Cₐ + Cᵦ)                         │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### Compound Partnership

```
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│  Partners invest for DIFFERENT durations                            │
│  Profit ratio = Capital × Time ratio                                │
│                                                                     │
│  A : B = Cₐ × Tₐ : Cᵦ × Tᵦ                                         │
│  A's share = Total Profit × (Cₐ×Tₐ) / Σ(C×T)                       │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### Working Partner

```
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│  Working partner gets SALARY first from total profit                │
│  Remaining profit is split by capital ratio                         │
│                                                                     │
│  Net profit for splitting = Total Profit - Salary                   │
│  Then divide net profit as per capital × time ratio                 │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### Capital Changes Mid-Year

```
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│  Split into periods at each capital change                          │
│  Effective capital = Σ(Cᵢ × Tᵢ)                                     │
│                                                                     │
│  A invests C₁ for t₁ months, then C₂ for t₂ months                  │
│  A's effective capital = C₁ × t₁ + C₂ × t₂                          │
│  Compare effective capitals for profit sharing ratio                │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Type 1: Simple Partnership

**Q: A and B invest Rs 3000 and Rs 5000 respectively for a year. If the total profit is Rs 1600, find each person's share.**

```
  Same duration → Simple Partnership
  A : B = 3000 : 5000 = 3 : 5

  Total ratio parts = 3 + 5 = 8
  A's share = 1600 × 3/8 = Rs 600
  B's share = 1600 × 5/8 = Rs 1000

  A ═══ Rs 3000 ──→ 3 parts ──→ Rs 600
  B ═════ Rs 5000 ──→ 5 parts ──→ Rs 1000
  |──────── Profit = 1600 ────────|

  CHECK: 600 + 1000 = 1600 ✓

  Answer: A = Rs 600, B = Rs 1000 ✓
```

## Type 2: Compound Partnership

**Q: A invests Rs 4000 for 12 months and B invests Rs 6000 for 8 months. If the total profit is Rs 2400, find their shares.**

```
  Different durations → Compound Partnership
  A = 4000 × 12 = 48000
  B = 6000 × 8  = 48000

  Ratio = 48000 : 48000 = 1 : 1
  A's share = 2400 / 2 = Rs 1200
  B's share = 2400 / 2 = Rs 1200

  A: 4000 × 12 months = 48000
  B: 6000 × 8 months  = 48000
  Ratio = 1:1 → Equal split

  CHECK: 1200 + 1200 = 2400 ✓

  Answer: A = Rs 1200, B = Rs 1200 ✓
```

## Type 3: Two Partners (Find Profit Share)

**Q: A and B start a business with Rs 20000 and Rs 25000. At the end of the year, the total profit is Rs 9000. Find B's share.**

```
  Same duration → Capital ratio
  A : B = 20000 : 25000 = 4 : 5

  Total parts = 4 + 5 = 9
  B's share = 9000 × 5/9 = Rs 5000
  A's share = 9000 × 4/9 = Rs 4000

  A ════ Rs 20000 → 4 parts
  B ═════ Rs 25000 → 5 parts

  CHECK: 4000 + 5000 = 9000 ✓

  Answer: B's share = Rs 5000 ✓
```

## Type 4: Three Partners

**Q: A, B, and C invest Rs 10000, Rs 15000, and Rs 20000 for the same period. Total profit is Rs 9000. Find each share.**

```
  A : B : C = 10000 : 15000 : 20000
            = 2 : 3 : 4

  Total parts = 2 + 3 + 4 = 9
  A = 9000 × 2/9 = Rs 2000
  B = 9000 × 3/9 = Rs 3000
  C = 9000 × 4/9 = Rs 4000

  A ══ Rs 10k → 2 parts → Rs 2000
  B ═══ Rs 15k → 3 parts → Rs 3000
  C ════ Rs 20k → 4 parts → Rs 4000

  CHECK: 2000 + 3000 + 4000 = 9000 ✓

  Answer: A = 2000, B = 3000, C = 4000 ✓
```

## Type 5: Working Partner with Salary

**Q: A and B invest Rs 5000 each. A is working partner and gets Rs 100/month as salary. Annual profit is Rs 2000. Find their shares.**

```
  A's annual salary = 100 × 12 = Rs 1200

  Remaining profit = 2000 - 1200 = Rs 800
  Capital ratio = 5000 : 5000 = 1 : 1
  Each gets 800/2 = Rs 400

  A's total = 1200 + 400 = Rs 1600
  B's total = Rs 400

  Total Profit ═══ Rs 2000
  ├─ A's salary ── Rs 1200
  └─ Remaining ─── Rs 800
     ├─ A (1:1) ── Rs 400
     └─ B (1:1) ── Rs 400
  A total = 1200 + 400 = 1600
  B total = 400

  Answer: A = Rs 1600, B = Rs 400 ✓
```

## Type 6: Partner Joins Late

**Q: A starts a business with Rs 8000. After 4 months, B joins with Rs 12000. At the end of the year, profit is Rs 5400. Find their shares.**

```
  A invests for 12 months
  B invests for 12 - 4 = 8 months

  A = 8000 × 12  = 96000
  B = 12000 × 8  = 96000
  Ratio = 1 : 1

  Each gets 5400/2 = Rs 2700

  Month: 1──────4──────────12
  A:     |═══════════════════| 8k × 12
  B:         |═══════════════| 12k × 8

  96000 : 96000 = 1 : 1

  Answer: A = Rs 2700, B = Rs 2700 ✓
```

## Type 7: Partner Leaves Early

**Q: A and B invest Rs 6000 and Rs 8000. A withdraws after 8 months. Annual profit is Rs 2380. Find their shares.**

```
  A invests for 8 months
  B invests for 12 months

  A = 6000 × 8  = 48000
  B = 8000 × 12 = 96000
  Ratio = 48000 : 96000 = 1 : 2

  Total parts = 3
  A = 2380 × 1/3 ≈ Rs 793.33
  B = 2380 × 2/3 ≈ Rs 1586.67

  Month: 1────────8────────12
  A:     |════════| 6k × 8 = 48k
  B:     |═════════════════| 8k × 12 = 96k

  Ratio = 1 : 2

  Answer: A ≈ Rs 793, B ≈ Rs 1587 ✓
```

## Type 8: Capital Changes Mid-Year

**Q: A starts with Rs 10000. After 6 months, A adds Rs 5000 more. B invests Rs 15000 for the whole year. Profit is Rs 3500. Find shares.**

```
  A's effective capital:
  10000 × 6 + 15000 × 6 = 60000 + 90000 = 150000

  B's effective capital:
  15000 × 12 = 180000

  Ratio = 150000 : 180000 = 5 : 6
  Total parts = 11

  A = 3500 × 5/11 ≈ Rs 1590.91
  B = 3500 × 6/11 ≈ Rs 1909.09

  Month: 1──────6──────────12
  A:     |═10k══|═══15k════|
  B:     |═══════15k═══════|

  A: 60k + 90k = 150k
  B: 180k → Ratio 5:6

  Answer: A ≈ Rs 1591, B ≈ Rs 1909 ✓
```

## Type 9: Find Investment Amount

**Q: A invests Rs 6000. B's profit share from a total profit of Rs 1500 is Rs 900. How much did B invest if both invested for the same period?**

```
  B's share = Rs 900
  A's share = 1500 - 900 = Rs 600

  Ratio of profits = A : B = 600 : 900 = 2 : 3

  Same duration → capital ratio = 2 : 3
  A/B = 2/3
  6000/B = 2/3
  B = 6000 × 3/2 = Rs 9000

  Profit: A = 600, B = 900 → 2 : 3
  Capital: A = 6000, B = ?
  6000/B = 2/3
  B = Rs 9000

  Answer: B invested Rs 9000 ✓
```

## Type 10: Monthly vs Annual Investment

**Q: A invests Rs 500 per month for 12 months. B invests Rs 3000 at the start for the whole year. Profit is Rs 1200. Find shares.**

```
  A's effective capital:
  500×12 + 500×11 + ... + 500×1
  = 500 × (12+11+...+1)
  = 500 × 78 = 39000

  B's effective capital:
  3000 × 12 = 36000

  Ratio = 39000 : 36000 = 13 : 12
  Total parts = 25

  A = 1200 × 13/25 = Rs 624
  B = 1200 × 12/25 = Rs 576

  A: Rs 500/month (cumulative monthly)
  B: Rs 3000 lump sum for 12 months
  39000 : 36000 = 13 : 12

  Answer: A = Rs 624, B = Rs 576 ✓
```
