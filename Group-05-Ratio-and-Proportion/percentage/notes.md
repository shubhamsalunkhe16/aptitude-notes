## ALL FORMULAS AT A GLANCE

### Basic Percentage Formulas

```
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│  x% of y = (x/100) × y                                             │
│                                                                     │
│  What % is X of Y = (X/Y) × 100                                    │
│                                                                     │
│  Percentage Increase = [(New - Old) / Old] × 100                    │
│  Percentage Decrease = [(Old - New) / Old] × 100                    │
│                                                                     │
│  New Value = Old × (1 ± %/100)                                      │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### Successive & Compound Change Formulas

```
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│  Successive change of a% and b%:                                    │
│     Net = a + b + (a×b)/100  %                                      │
│                                                                     │
│  Population after n years (growth r%):                              │
│     P × (1 + r/100)^n                                               │
│                                                                     │
│  Depreciation after n years (decline r%):                           │
│     P × (1 - r/100)^n                                               │
│                                                                     │
│  Two successive discounts a% and b%:                                │
│     Single equiv = a + b - (a×b)/100  %                             │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### Percentage Change & Reversal Formulas

```
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│  If A is x% more than B:                                            │
│     B is less than A by [x/(100+x)] × 100 %                        │
│                                                                     │
│  If A is x% less than B:                                            │
│     B is more than A by [x/(100-x)] × 100 %                        │
│                                                                     │
│  If price ↑ x%, to keep spending same:                              │
│     Reduce consumption by [x/(100+x)] × 100 %                      │
│                                                                     │
│  If price ↓ x%, to keep spending same:                              │
│     Increase consumption by [x/(100-x)] × 100 %                    │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### Area Change & Exam/Election Formulas

```
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│  Length ↑ a%, Breadth ↑ b%:                                         │
│     Area change = a + b + (a×b)/100  %                              │
│     (Same as successive change formula!)                            │
│     Use -ve for decrease                                            │
│                                                                     │
│  Pass marks = Obtained + marks short of passing                     │
│  Pass marks = Total × (pass% / 100)                                │
│                                                                     │
│  Election: Winner gets W% of votes                                  │
│     Winner - Loser = Margin                                         │
│     Winner% + Loser% = 100%  (2 candidates)                        │
│     Margin% = (2W - 100)%                                          │

│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## KEY CONCEPT — Percentage as a Fraction of 100

```
         Percentage
        /          \
       /            \
   "Per cent"    = out of 100

   x% = x/100  (fraction)
       = x × 0.01  (decimal)

Step 1: Identify what is the BASE (denominator)
        "X is what % of Y" → Y is the base
Step 2: Apply formula: (Part / Whole) × 100
Step 3: For change: (Change / Original) × 100

   GOLDEN RULE:
   % is ALWAYS relative to some base.
   "A is 25% more than B" → base is B
   "A is 25% less than B" → base is B
   "A is what % of B" → base is B
```

## Type 1: Basic Percentage (What is x% of y)

**Q: What is 15% of 840?**

```
  15% of 840

  Method 1: Direct
    = (15/100) × 840
    = 0.15 × 840
    = 126 ✓

  Method 2: Split
    10% of 840 = 84
     5% of 840 = 42  (half of 10%)
    ─────────────────
    15% of 840 = 126 ✓

  Method 3: Commutative trick
    15% of 840 = 840% of 15 ← harder
    Better: break 840 into easy parts
    15% of 800 = 120
    15% of 40  = 6
    Total = 126 ✓
```

## Type 3: Percentage Increase / Decrease

**Q: A salary increases from Rs.25000 to Rs.30000. Find % increase.**

```
  Increase = New - Old = 30000 - 25000 = 5000

  % Increase = (Increase / Old) × 100
             = (5000 / 25000) × 100
             = (1/5) × 100
             = 20% ✓

  BEFORE ──────────────────► AFTER
  25000 ──── +20% ─────────► 30000

  NOTE: Base is ALWAYS the original (old) value

  TRAP: If salary decreases from 30000 to 25000:
  % Decrease = (5000 / 30000) × 100
             = 16.67% (NOT 20%!)
        base changes → answer changes!
```

## Type 4: Successive Percentage Change

**Q: Price rises by 10% then by 20%. Net % increase?**

```
  Formula: Net = a + b + (a × b)/100

  a = 10%, b = 20%

  Net = 10 + 20 + (10 × 20)/100
      = 10 + 20 + 200/100
      = 10 + 20 + 2
      = 32% increase ✓

  Verification with Rs.100:
    100 ──+10%──► 110 ──+20%──► 132
    Net change = 132 - 100 = 32 ✓

  SPECIAL CASES:
  ┌──────────────────────────────────────┐
  │ +10% then -10% = -1% (NOT zero!)    │
  │ +20% then -20% = -4% (NOT zero!)    │
  │ +x% then -x% = -(x²/100)% ALWAYS   │
  │                    → always a LOSS!  │
  └──────────────────────────────────────┘
```

## Type 5: Population Growth / Depreciation (Compound)

**Q: Population of town is 50000. It grows at 10% per annum. Population after 3 years?**

```
  Formula: P_final = P × (1 + r/100)^n

  P = 50000,  r = 10%,  n = 3

  = 50000 × (1 + 10/100)^3
  = 50000 × (1.1)^3
  = 50000 × 1.331
  = 66550 ✓

  Year by year:
  Year 0: 50000
  Year 1: 50000 × 1.1 = 55000
  Year 2: 55000 × 1.1 = 60500
  Year 3: 60500 × 1.1 = 66550 ✓

  FOR DEPRECIATION (value decreases):
  V_final = V × (1 - r/100)^n

  Machine Rs.80000, depreciates 10%/yr, after 2 yrs:
  = 80000 × (0.9)^2 = 80000 × 0.81 = 64800
```

## Type 6: Price Increase → Consumption Decrease (Same Spending)

**Q: Price of sugar rises by 25%. By what % must a family reduce consumption so that spending stays the same?**

```
  Formula: Reduce by [x/(100+x)] × 100 %

  x = 25%

  = [25 / (100+25)] × 100
  = [25/125] × 100
  = (1/5) × 100
  = 20% reduction ✓

  Verification:
    Let price = 100, qty = 100  → spend = 10000
    New price = 125, new qty = ?
    125 × qty = 10000 → qty = 80
    Reduction = (100-80)/100 × 100 = 20% ✓

  REVERSE: Price ↓ 20%, consumption ↑ by?
  = [20/(100-20)] × 100 = 20/80 × 100 = 25%

  ┌─────────────────────────────────────────────┐
  │ Price ↑ 25%  →  Consumption ↓ 20%           │
  │ Price ↑ 20%  →  Consumption ↓ 16.67%        │
  │ Price ↑ 10%  →  Consumption ↓ 9.09%         │
  │ Price ↑ 50%  →  Consumption ↓ 33.33%        │
  │ Price ↑ 100% →  Consumption ↓ 50%           │
  └─────────────────────────────────────────────┘
```

## Type 7: Two Successive Discounts

**Q: A shopkeeper gives two successive discounts of 20% and 10%. Find single equivalent discount.**

```
  Formula: Single discount = a + b - (a×b)/100

  a = 20%, b = 10%

  = 20 + 10 - (20 × 10)/100
  = 30 - 200/100
  = 30 - 2
  = 28% single discount ✓

  NOTE: Minus sign (NOT plus like successive increase)
        Because discounts REDUCE the price.

  Verification with Rs.100:
    100 ──-20%──► 80 ──-10%──► 72
    Single discount = 100 - 72 = 28% ✓

  TRAP: 20% + 10% ≠ 30% discount!
        Two 50% discounts ≠ 100% off (it's 75%)

  ┌─────────────────────────────────────────────┐
  │ Order does NOT matter for net discount       │
  │ 20% then 10% = 10% then 20% = 28%           │
  │ BUT: 3 discounts: a+b+c-ab/100-bc/100       │
  │      -ac/100+abc/10000 (use step by step!)   │
  └─────────────────────────────────────────────┘
```

## Type 8: Percentage Change in Area (Length + Breadth Change)

**Q: If length of a rectangle increases by 20% and breadth decreases by 10%, find % change in area.**

```
  Area = Length × Breadth
  This is successive change on area!

  Formula: Net = a + b + (a×b)/100

  a = +20% (increase), b = -10% (decrease)

  = 20 + (-10) + (20 × (-10))/100
  = 20 - 10 - 200/100
  = 10 - 2
  = +8% (area increases by 8%) ✓

  Verification:
    Let L=100, B=100 → Area = 10000
    New L=120, B=90  → Area = 10800
    Change = (10800-10000)/10000 × 100 = 8% ✓

  SPECIAL CASE: Both sides increase by x%
    Area change = 2x + x²/100
    e.g. both ↑10%: net = 20+1 = 21% area increase

  SIDE OF SQUARE ↑ x%:
    Area ↑ by (2x + x²/100)%
    Side ↑10% → Area ↑21%
    Side ↑20% → Area ↑44%
```

## Type 9: Pass Marks Problem (Exam Pass/Fail Percentage)

**Q: A student gets 210 marks and fails by 40 marks. Another student gets 350 marks which is 30 more than pass marks. Find pass marks and total marks.**

```
  Student 1: Gets 210, fails by 40
    → Pass marks = 210 + 40 = 250

  Student 2: Gets 350, 30 more than pass marks
    → Pass marks = 350 - 30 = 320

  WAIT — both must give same pass marks!

  CORRECT approach when pass% is given:
  Q: Gets 40% and fails by 30 marks. Gets 60% and has
     20 more than pass marks. Find total marks.

  40% of Total = Pass - 30  ─── (i)
  60% of Total = Pass + 20  ─── (ii)

  (ii) - (i):
  60% - 40% = (Pass+20) - (Pass-30)
  20% of Total = 50
  Total = 50 × 100/20 = 250 marks ✓
  Pass marks = 40% of 250 + 30 = 100 + 30 = 130 ✓

  Visual:
  |════════════════════════════════════|
  0%          40%    Pass    60%     100%
  |           |──30──|       |        |
  |           100    130     150      250
  |                  ↑       |──20──►|
              fails by 30    exceeds by 20
```

## Type 10: Salary Percentage Problems (A earns x% more than B)

**Q: A's salary is 20% more than B's. By what % is B's salary less than A's?**

```
  Formula: B is less by [x/(100+x)] × 100 %

  x = 20%

  = [20/(100+20)] × 100
  = [20/120] × 100
  = (1/6) × 100
  = 16.67% ✓

  Verification:
    Let B = 100, then A = 120
    B is less than A by = (120-100)/120 × 100
                        = 20/120 × 100 = 16.67% ✓

  NOTE: Different base → different answer!

  ┌──────────────────────────────────────────────┐
  │  A is 20% more than B                        │
  │  → B is NOT 20% less than A                  │
  │  → B is 16.67% less than A                   │
  │                                               │
  │  If A is x% more than B:                      │
  │    B less by x/(100+x) × 100                  │
  │                                               │
  │  If A is x% less than B:                      │
  │    B more by x/(100-x) × 100                  │
  └──────────────────────────────────────────────┘

  Common pairs:
  ┌─────────────────────────────────────┐
  │ A > B by   │ B < A by              │
  │ 10%        │ 9.09%  (10/110)       │
  │ 20%        │ 16.67% (20/120 = 1/6) │
  │ 25%        │ 20%    (25/125 = 1/5) │
  │ 33.33%     │ 25%    (1/3 of 4/3)   │
  │ 50%        │ 33.33% (50/150 = 1/3) │
  │ 100%       │ 50%    (100/200)      │
  └─────────────────────────────────────┘
```

## Type 14: If A is x% More Than B, B is What % Less Than A

**Q: If A is 25% more than B, then B is what % less than A?**

```
  Formula: B is less by [x / (100 + x)] × 100 %

  x = 25%

  = [25 / 125] × 100
  = (1/5) × 100
  = 20% ✓

  Proof:
    Let B = 100 → A = 125
    Diff = 25
    % less = 25/125 × 100 = 20% ✓
    (base is A since "B is less than A")

  ┌─────────────────────────────────────────────────────┐
  │  A > B by x%  ⇒  B < A by [x/(100+x)] × 100 %     │
  │  A < B by x%  ⇒  B > A by [x/(100-x)] × 100 %     │
  │                                                     │
  │  Example: A is 20% less than B                      │
  │  B is more than A by:                               │
  │  = [20/(100-20)] × 100 = 20/80 × 100 = 25%         │
  └─────────────────────────────────────────────────────┘

  VISUAL:
    B = 100 ──────────────────────
    A = 125 ──────────────────────────
              A is 25% MORE than B (base B=100)
              B is 20% LESS than A (base A=125)

    DIFFERENT BASE → DIFFERENT ANSWER!
```

## Type 15: Election / Voting Problems

**Q: In an election, candidate A gets 60% of votes. He wins by 8000 votes. Find total votes.**

```
  A gets 60% → B gets 40% (two candidates)

  Margin = A% - B% = 60% - 40% = 20%

  20% of Total = 8000
  Total = 8000 × 100/20 = 40000 votes ✓

  A got = 60% of 40000 = 24000 ✓
  B got = 40% of 40000 = 16000 ✓
  Diff = 24000 - 16000 = 8000 ✓

  VARIATION with invalid votes:
  Q: 20% votes invalid. A gets 60% of valid votes.
     Wins by 4800. Total votes?

  Valid votes = 80% of Total
  A = 60% of valid, B = 40% of valid
  Margin = 20% of valid = 4800
  Valid = 24000
  Total = 24000 / 0.8 = 30000 ✓

  ┌──────────────────────────────────────────┐
  │  Total Votes: 30000                      │
  │  ├── Invalid: 20% = 6000                 │
  │  └── Valid:   80% = 24000                │
  │       ├── A: 60% of 24000 = 14400       │
  │       └── B: 40% of 24000 = 9600        │
  │  Margin = 14400 - 9600 = 4800 ✓         │
  └──────────────────────────────────────────┘
```
